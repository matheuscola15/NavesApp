Tutorial

Passo a passo

A primeira coisa que devemos fazer é a instalação do Node.js, e instale a versão LTS

Link: https://nodejs.org/en/

Após a instalação do node, iremos ir para o prompt de comando de seu computador para fazer a instalação do Ionic CLI, digitaremos o seguinte código dentro do prompt:

npm install -g @ionic/cli

npm uninstall -g ionic

npm install -g @ionic/cli

A parte de instalação do Ionic está completa, agora iremos para a criação do seu projeto! 1

Ainda dentro do seu prompt, vamos criar seu projeto:

ionic start myApp tabs -–type=angular

O ionic start é para a criarmos o projeto

O myApp é apenas o nome do seu projeto, ele pode ser modificado por você como quiser!

O tabs é o template que igualmente pode ser mudado por você, temos as seguintes opções de template: tabs, blank e sidemenu

E o -–type=angular é basicamente o tipo do projeto que iremos trabalhar, que será em angular, mas fica a sua escolha com o que você quer trabalhar.

Assim, temos nosso projeto ionic, agora é usar a sua genialidade e criatividade dentro do projeto!
